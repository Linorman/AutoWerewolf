## 一、基础规则

### 1. 阵营划分
- **狼人阵营**：目标是通过夜晚杀人或白天混淆视听，达成胜利条件。
- **好人阵营**：包括**村民**（无技能）和**神民**（有特殊技能），共同目标是找出并淘汰所有狼人。

### 2. 胜利条件
- **好人胜利**：必须杀掉所有狼人。
- **狼人胜利**（常见为“屠边局”）：
  - 杀光所有**神民**，或
  - 杀光所有**村民**。
  - （有时也采用“屠城局”：杀光所有好人）
  - 若白天投票环节中狼人票数 ≥ 好人票数，也可直接判狼人胜。

### 3. 遗言规则
- **白天被投死**：可发表遗言（所有天都可）。
- **夜晚被杀**：**仅第一夜**死亡者可发表遗言；之后夜晚死亡者**无遗言**。

### 4. 警长竞选（仅第一天白天）
- 想当警长者举手 → 依次发言 → 未参选者投票。
- 警长功能：
  - 投票为1.5票；
  - 决定发言顺序（死左/死右 或 警左/警右）；
  - 归票（推荐投谁）；
  - 死亡时可传警徽或撕警徽。

## 二、游戏角色（12人标准局）

### 狼人阵营（4人）
- **狼人**：
  - 每晚睁眼，共同刀杀一人；
  - 白天可自爆（直接进入黑夜，自己死亡）；
  - 术语：“刀”（杀人）、“自刀”（狼杀狼骗药/做身份）。

### 好人阵营（8人）

#### 村民（4人）
- 无技能，仅靠白天发言和投票推理。

#### 神民（4人，两种常见配置）

| 角色 | 技能 | 术语 |
|------|------|------|
| **预言家** | 每晚验一人身份（好人/狼人） | **金水**（验出好人）<br>**查杀**（验出狼人） |
| **女巫** | 有一瓶解药（救人）、一瓶毒药（毒人）<br>每晚最多用一种药，不可自救（部分局允许首夜自救） | **银水**（被救的人，通常视为好人） |
| **猎人** | 被投票或狼杀时可开枪带走一人（被女巫毒死则不能开枪） | — |
| **守卫**（或**白痴**） | **守卫**：每晚守护一人（不能连守同一人）<br>**白痴**：白天被投出可亮身份免死（失去投票权，但可插话） | **同守同救**：守卫守+女巫救同一人 → 仍死亡 |

> 常见神职组合：
> - 预言家 + 女巫 + 猎人 + 守卫  
> - 预言家 + 女巫 + 猎人 + 白痴

## 三、游戏流程（循环进行）

### 第一夜
- 狼人睁眼杀人；
- 各神民依次睁眼发动技能。

### 第一天白天
1. 警长竞选；
2. 上帝宣布第一夜死讯；
3. **第一夜死者发表遗言**；
4. 所有人依次发言；
5. 投票放逐一人；
6. **被放逐者发表遗言**。

### 第二夜及以后
- **黑夜**：狼人杀人 → 存活神民发动技能。
- **白天**：宣布死讯 → 发言 → 投票 → 死者遗言。

> 流程循环，直至某一阵营达成胜利条件。
